# Usage
## Installation
```Julia
using Pkg
Pkg.add("HCTSA")
using HCTSA
```

> [!WARNING]
> Please set `JULIA_COPY_STACKS=1` in your environment before starting a VSCode session if you are using HCTSA.jl with the Julia VSCode extension. You can do this by adding `export JULIA_COPY_STACKS=1` to your `.bashrc` or equivalent shell configuration file.
